# AWS Skill Builder — Recommended Courses

General-purpose AWS Skill Builder courses for upskilling Devs and QAs.
All courses are free unless noted. Accessible at [explore.skillbuilder.aws](https://explore.skillbuilder.aws).

> **Tip:** The Skill Builder portal uses dynamic URLs. Search by the exact course name on the portal to find each course.

---

## Level 1 — Cloud Foundations

Start here for anyone new to AWS.

### 1. AWS Cloud Practitioner Essentials

The most popular foundational course. Covers cloud concepts, core services, security, architecture, pricing.

> **Note for QAs:** Take this course as your primary cloud foundation. You can skip Course #2 (Cloud Quest) to avoid redundancy and save ~12 hours.

⏱ **~6 hours**

- Landing page: [aws.amazon.com/training/classroom/aws-cloud-practitioner-essentials](https://aws.amazon.com/training/classroom/aws-cloud-practitioner-essentials/)
- Also on Coursera: [coursera.org/learn/aws-cloud-practitioner-essentials](https://www.coursera.org/learn/aws-cloud-practitioner-essentials)
- On Skill Builder: search **"AWS Cloud Practitioner Essentials"**

### 2. AWS Cloud Quest: Cloud Practitioner

Game-based learning, very engaging. Builds practical skills through interactive challenges in a live AWS environment. Free tier available.

> **Note for QAs:** Optional alternative to Course #1 if you prefer gamified learning over lecture-style. Do not take both.

⏱ **~12 hours** (self-paced, varies by player)

- Info: [aws.amazon.com/training/learn-about/cloud-practitioner](https://aws.amazon.com/training/learn-about/cloud-practitioner/)
- On Skill Builder: search **"Cloud Quest Cloud Practitioner"**

### 3. AWS Builder Labs Learning Plan

10 guided hands-on labs covering VPC, S3, EC2, IAM, KMS, Lambda, DynamoDB, and more. Real AWS console, no personal account needed.

⏱ **~8-10 hours** (10 labs, ~45-60 min each)

- Announcement: [aws.amazon.com/blogs/training-and-certification/begin-your-aws-journey-with-new-free-aws-builder-labs-learning-plan-on-aws-skill-builder](https://aws.amazon.com/blogs/training-and-certification/begin-your-aws-journey-with-new-free-aws-builder-labs-learning-plan-on-aws-skill-builder/)

## Level 2 — Security & IAM

### 4. AWS Security Fundamentals (Second Edition)

Covers access control, encryption, governance, logging.

⏱ **~2 hours**

- On Skill Builder: search **"AWS Security Fundamentals Second Edition"**
- Reference: [classcentral.com/course/aws-security-fundamentals-second-edition-271288](https://www.classcentral.com/course/aws-security-fundamentals-second-edition-271288)

### 5. AWS Security Essentials

Deeper dive into shared responsibility model, data encryption, network security.

⏱ **~8 hours** (1-day classroom equivalent, self-paced)

- Landing page: [aws.amazon.com/training/classroom/aws-security-essentials](https://aws.amazon.com/training/classroom/aws-security-essentials/)

### 6. AWS Security Champion Knowledge Path

Comprehensive path for developers wanting security skills. Covers IAM troubleshooting, encryption, threat modeling, incident response, container security. Earns a digital badge. **Requires Skill Builder subscription ($29/month individual).**

⏱ **~15-20 hours** (multi-module path with hands-on labs and SimuLearn)

- Announcement: [aws.amazon.com/blogs/security/introducing-the-aws-security-champion-knowledge-path-and-digital-badge](https://aws.amazon.com/blogs/security/introducing-the-aws-security-champion-knowledge-path-and-digital-badge/)

## Level 3 — Serverless & Application Services

### 7. Serverless Learning Plan

Curated path covering Lambda foundations, API Gateway, architecting serverless apps, scaling, security & observability, deployment, and event-driven architectures (Step Functions, SQS). Earns a digital badge upon completion. Free.

> **QA Focus (~5-6 hours):** Focus on **Lambda foundations**, **API Gateway**, and **event-driven architectures (SQS, Step Functions)** to understand async event triggers, payloads, and DLQs. Skip/skim advanced architecture, deployment, and scaling modules.

⏱ **~10-12 hours** (7 courses + assessment; ~5-6 hours for QA focus)

- Info: [aws.amazon.com/training/learn-about/serverless](https://aws.amazon.com/training/learn-about/serverless/)
- Blog: [aws.amazon.com/blogs/compute/introducing-new-aws-serverless-digital-learning-badges](https://aws.amazon.com/blogs/compute/introducing-new-aws-serverless-digital-learning-badges/)
- On Skill Builder: search **"Serverless Learning Plan"**

### 8. Amazon DynamoDB for Developers (3-course series)

Covers interacting with tables programmatically, data modeling, and advanced operations.

> **QA Focus (~2-3 hours):** Take Course 1 (interacting with tables, querying, scanning, and viewing items in Console/CLI for test verification). Skip advanced data modeling (single-table design) and complex programmatic transactions.

⏱ **~6-8 hours** (~90-150 min per course × 3 courses; ~2-3 hours for QA focus)

- Blog: [aws.amazon.com/blogs/training-and-certification/accelerate-your-application-development-skills-with-three-free-amazon-dynamodb-courses](https://aws.amazon.com/blogs/training-and-certification/accelerate-your-application-development-skills-with-three-free-amazon-dynamodb-courses/)
- On Skill Builder: search **"Amazon DynamoDB"**

## Level 4 — Architecture & Observability

### 9. AWS Well-Architected for Enterprises

Covers the 6 pillars (security, reliability, performance, cost, operational excellence, sustainability). Interactive scenarios for identifying and remediating architecture risks.

⏱ **~1 hour**

- Blog: [aws.amazon.com/blogs/training-and-certification/well-architected-for-enterprises](https://aws.amazon.com/blogs/training-and-certification/well-architected-for-enterprises/)
- On Skill Builder: search **"AWS Well-Architected"**

### 10. AWS Observability (4-part course)

CloudWatch Logs, Metrics, Alarms, X-Ray tracing, dashboards. Very useful for anyone debugging production issues.

⏱ **~5 hours**

- Blog: [aws.amazon.com/blogs/mt/build-cloud-operations-skills-using-the-new-aws-observability-training](https://aws.amazon.com/blogs/mt/build-cloud-operations-skills-using-the-new-aws-observability-training/)
- On Skill Builder: search **"AWS Observability"**

## Bonus — Certification Prep (optional)

### 13. AWS Certified Cloud Practitioner

Validate foundational knowledge with a cert. Exam: 65 questions, 90 minutes.

⏱ **~40-80 hours total study** (2-6 weeks depending on background)

- [aws.amazon.com/certification/certified-cloud-practitioner](https://aws.amazon.com/certification/certified-cloud-practitioner/)

### 14. AWS Certified Solutions Architect – Associate

For devs who want to go deeper into architecture.

⏱ **~80-120 hours total study** (2-3 months recommended)

- On Skill Builder: search **"Solutions Architect Associate"**
